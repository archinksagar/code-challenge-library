# Coding Challenge - Library

Designing and implementing a REST API for Library management.  

# Project Name : Library

## Overview
REST API provides the following end points.

### /users/
### /users/{user_id}/
### /transactions/
### /transactions/{transaction_id}/
### /books/
### /books/{book_id}/
### /users/{user_id}/books/
### /library/
### /library/{library_id}/books/

### Technologies 
Django 3.1.5
Python 3.9.1
RestFramework
SQLLite
